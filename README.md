# Smart-Task---Productivity-Manager-with-java
TaskFlow is a modern full stack productivity and task management web application designed to help users efficiently organize, track, and manage their daily tasks and work progress through an interactive dashboard interface.

The platform allows users to create, update, delete, prioritize, and monitor tasks while improving productivity through visual analytics and task completion tracking. Users can organize tasks based on priority levels, due dates, and completion status, making the application suitable for personal productivity management and workflow organization.

The system includes a secure user authentication module with login and signup functionality, ensuring personalized task management for each user. A responsive dashboard provides detailed productivity insights including completed tasks, pending tasks, overdue tasks, and task completion statistics using interactive charts and progress indicators.

TaskFlow also includes search and filtering functionality, allowing users to quickly find tasks based on priority, status, or deadlines. The application supports both Day Theme and Night Theme modes with smooth theme transition effects to provide a modern and user-friendly experience.

The project is developed using Core Java for backend logic, SQLite for lightweight database management, and HTML, CSS, JavaScript, and Bootstrap for frontend development. JDBC is used for database connectivity and CRUD operations.

The application is designed with a modern SaaS-inspired UI while maintaining beginner-friendly architecture and simple understandable code structure suitable for fresher-level full stack development projects.
